Traformazioni puntuali

Le trasformazioni puntuali sono legate dalla relazione s = T(r) dove T(r) è una trasformazione che mappa il valore di r di un pixel nel corrispondente valore s.

Negativo dell'immagine

Il negativo di un immagine con livelli di intensità nella gamma [0, L-1] si ottiene utilizzando la trasformazione negativa che è data dall'equazione:

s = L - 1 - r
Invertendo i livelli di intensità si genera l'equivalente di un negativo fotografico.

Trasformazioni logaritmiche

La forma generale della trasformazione logaritmica è

s = c log (r + 1)

dove c è una costante e si presuppone che r ≥ 0.

La forma della curva logaritmica mostra che questa trasformazione associa una stretta gamma di valori a bassa intensità una gamma più ampia di valori di output. Si verifica il contrario con valori più alti dei livelli di input. La funzione logaritmica ha l'importante caratteristica di comprimere la gamma dinamica delle immagini che presentano ampie variazioni dei valori dei pixel. Un classico esempio è lo spettro di Fourier.

Oltre alla trasformazione logaritmo è possibile impiegare la trasformazione logaritmica inversa nella quale, diversamente dalla trasformazione logaritmica, un'ampia gamma di di valori a bassa intensità è associata ad una gamma più stretta in ragione di una trasformazione logaritmica inversa.

Trasformazioni potenza o gamma

Le trasformazioni di potenza hanno la forma di base:

s = c r γ

dove c e γ sono costanti positive.

Come nel caso delle trasformazioni logaritmiche, le curve di potenza con i valori frazionali di γ trasformano una stretta gamma di valori scuri in una gamma più ampia di valori di output. Al contrario delle funzioni logaritmiche in questo caso è possibile ottenere un insieme di possibili curve semplicemente variando γ. Come è noto le curve generate con valori di γ >1 hanno esattamente l'effetto opposto a quelle generate con valori di γ < 1. Normalmente l'esponente è indicato con la lettera greca γ da cui discende il nome di trasformazione γ. Un approccio alternativo ai metodi presentati precedentemente è l'uso di funzioni lineari a tratti.

Amplitude Level Slicing

Assegnati due valori di soglia, bassa <(L) ed alta (H), si portano a bianco i livelli compresi fra esse.

I livelli rimanenti possono:

  • essere messi a zero, in questo caso si parla di Zero Background Level Slicing;
  • lasciati invariati con la funzione identità, in questo caso si parla di Image Background Level Slicing.
